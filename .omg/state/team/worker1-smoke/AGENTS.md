# worker1-smoke

## Purpose
Runtime snapshot directory for team namespace `worker1-smoke` capturing lifecycle transitions and latest verification status.

## Key Files

| File | Description |
|------|-------------|
| `phase.json` | Current lifecycle phase plus transition history for the latest run in this namespace. |
| `monitor-snapshot.json` | Latest runtime/team snapshot (when produced by the backend). |
