## Purpose
Operational metadata and agent-orchestration workspace generated by OMX tooling (plans, logs, drafts, runtime state, and operator preferences).

## Key Files

| File | Description |
|------|-------------|
| `metrics.json` | Session/usage counters and recent activity metrics. |
| `notepad.md` | Working-memory notes and checkpoints used across sessions. |
| `tmux-hook.json` | TMUX injection hook configuration (marker, cooldown, mode gates). |
